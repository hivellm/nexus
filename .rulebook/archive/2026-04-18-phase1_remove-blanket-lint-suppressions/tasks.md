## 1. Implementation
- [x] 1.1 Remove `all = "allow"` under `[lints.clippy]` from `nexus-server/Cargo.toml`, `nexus-core/Cargo.toml`, `nexus-cli/Cargo.toml`, `nexus-protocol/Cargo.toml`. `nexus-core` keeps a tight allow-list for three clippy lints (`too_many_arguments`, `type_complexity`, `result_large_err`) that fire in dozens of planner / executor hot-path call sites where fixing them would be churn with no signal — rationale is inlined in the Cargo manifest.
- [x] 1.2 Remove `unused = "allow"`, `unused_imports = "allow"`, `unused_variables = "allow"`, `unused_mut = "allow"`, `unused_assignments = "allow"`, and `dead_code = "allow"` from every `[lints.rust]` table. Test fixtures that keep `let (mut executor, ...)` preemptively get a file-scoped `#![allow(unused_mut)]` with a comment explaining the pattern; benchmark files behind `#[ignore]` tests get `#![allow(dead_code)]` for the same reason.
- [x] 1.3 Run `cargo clippy --workspace --all-targets -- -D warnings` and triage the new failures — 115 errors surfaced across nexus-core (unused_mut in test fixtures, a handful of hot-path `u64 as u64` casts, redundant imports, wrong-use-of-unwrap-after-is_ok, `clippy::format_in_format_args`, `clippy::length-comparison-to-one`, `clippy::approx_constant`, `clippy::collapsible_if`) plus 14 in nexus-server (MutexGuard across await in test helpers, missing `Default`, redundant closures, useless format!, `if let` refactor, unused imports) plus 15 in nexus-cli (unused imports, redundant closures, strip_prefix opportunities, identical-blocks collapse). Every one has been fixed or narrowly allowed with a rationale.
- [x] 1.4 For each triaged warning the fix was applied directly; the only surgical `#[allow(...)]` attributes added outside the Cargo lint tables are per-file `#![allow(unused_mut)]` on test fixtures that keep `let mut` preemptively, per-file `#![allow(dead_code)]` on `#[ignore]`-gated benchmark scaffolding, and attribute-level `#[allow(dead_code)]` on kept-for-parity API methods (e.g. `Config::get_profile` in `nexus-cli`). Each has a 1-line comment.
- [x] 1.5 Remove the `unused manifest key: example.1.num_cpus` warning from `nexus-core/Cargo.toml` — the stray `num_cpus = "1.16"` line under the second `[[example]]` was deleted.
